# politeia

**Politeia** — palavra grega para *constituição, regime, cidade* — é um repositório pessoal de leituras em teoria política. Cada módulo é uma página de leitura interativa, com conteúdo explicado de forma acessível e sistema de anotações por destaque de texto. Novos materiais são adicionados conforme novas leituras são trabalhadas.

Os temas cobertos incluem democracia e seus fundamentos, teorias do poder, pensamento institucional e filosofia política contemporânea. A plataforma cresce junto com as leituras.
